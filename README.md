# Social-Network-using-PHP-MySQL
Made as a DBMS Project, capable of:
1)Add friend/Remove Friend
2)Text Messaging 
3)Likes/Comments
4)Posting status/links/videos/pictures 
5)Deactivating and activating account 
6)Login/Logout
7)Register
8)Changing profile picture 

Consists of : 
1)Timeline
2)Newsfeed
3)Message window
